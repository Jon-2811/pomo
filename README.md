## 今回の追加内容

- 終了音を約2.4秒の、はっきりした2回チャイムへ変更
- 問題タイマーには区別しやすい短い2回警告音
- 設定画面に「終了音の音量」0〜100%スライダーを自動追加
- 「試聴」ボタンを追加
- iPhoneで最初のスタート操作時に音声を有効化
- ホーム画面版PWAで通知許可を要求
- Service Workerキャッシュをv3へ更新

## iPhoneでの確認

1. GitHubへ反映後、1〜2分待つ
2. Safariで公開ページを開き、再読み込みする
3. ホーム画面版Focus Glassを完全に閉じて開き直す
4. 設定 → 終了音の音量を100% → 試聴
5. タイマーを1分にしてスタートし、通知を許可する

アプリ内チャイムの音量はこのスライダーで調整できます。ロック画面に出るシステム通知音の音量は、iPhone本体の「着信音と通知音」の音量に従います。
