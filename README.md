# Focus Glass iPhone通知修正

## GitHubでの反映

1. `alerts.js` をリポジトリ直下に新規追加します。
2. 既存の `cloud.js` を同梱ファイルで置き換えます。
3. 既存の `sw.js` を同梱ファイルで置き換えます。
4. 3ファイルを同じコミットで保存します。

## iPhone側の更新

1. Safariで公開URLを一度開き、再読み込みします。
2. ホーム画面のFocus Glassを完全に閉じて、もう一度開きます。
3. タイマーの「スタート」を押します。
4. 通知許可が表示されたら「許可」を選びます。
5. まず1分に設定して、終了音とロック画面通知を確認します。

通知許可が表示されない場合は、Safariの共有メニューから改めて「ホーム画面に追加」し、ホーム画面のアイコンから起動してください。

## 重要な制約

この修正で、アプリを開いている時、ホーム画面版がバックグラウンドに残っている時、再度開いた時の通知・終了音を改善します。

ただし、iPhoneのAppスイッチャーでWebアプリを上に払って完全終了した状態では、静的なWebアプリだけで指定時刻に処理を起動できません。完全終了中にも確実に通知するには、Push APIへ通知を送るサーバー側処理が別途必要です。`sw.js` には将来のWeb Push受信処理を追加済みです。
